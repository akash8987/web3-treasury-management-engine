# Web3 Treasury Management Engine

A professional Web3 reference project that demonstrates how decentralized treasury systems can be modeled using clean and modular Web3 architecture.

This repository focuses on simulating treasury inflows, tracking fund allocations, validating spending rules, and generating financial analytics in a beginner-friendly and production-style format.

---

## Supported Networks

- Ethereum  
- Base  
- Polygon  
- Arbitrum  

---

## Project Goals

- Simulate treasury inflows and outflows  
- Track fund allocation categories  
- Validate spending limits  
- Analyze treasury performance  
- Teach DAO finance architecture  

---

## Architecture Overview

This project uses a root-only structure where each file handles a single responsibility such as network access, treasury registry, allocation engine, spending validator, and analytics reporting.

---

## Features

- Multi-chain RPC support  
- Treasury simulation engine  
- Fund allocation tracking  
- Spending rule validation  
- Clean console-based reporting
